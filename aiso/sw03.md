# SW03 - Rational Agents

## What is a rational agent?

An agent is a *thing* (e.g. a piece of software or hardware) with sensors and actuators. Its rationality lies within the goal, to maximize its performance measure.

The rational agent acts so as to achieve the best (expected) outcome.

### Agents

* Make decisions based on observations
* **Agent functions** map sequence of observations (percepts) to actions
* Can influence environment through actions
* Actions might fail to have desired effect
* -> Agents need to deal with possibility of failure
